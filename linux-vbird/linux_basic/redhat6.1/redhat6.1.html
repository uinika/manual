简易基础篇
章节标题内容大纲
写在前面 一开始总是要对 Linux 的起源做一些介绍的吧！然后再建议 Linux 的学习方法等等的。Linux 最早是由 Linus Torvalds 所写出的第一套类 Unix 的核心程序， 经由不断的开发才有今日的规模。不过，由于于开发者众多，所以导致有很多不同的 distribution 版本！.....
安装之前 是否准备要安装 Linux 了呢？别急别急，总是得知道一下你所要安装 Linux 的硬件（硬盘、CPU、RAM尤其是 VGA）是否可以被 Linux 的核心支持呢？所以你得先了解何种硬设备适合 Linux？如何规划硬盘？如何取得 Linux 光盘.....
简易的安装程序 好了，开始来安装吧！实地进行硬盘划分，本文并以 Windows98 及 Linux 并存的架构进行 Linux 的安装，同时并以 SPFdisk 介绍硬盘划分的方法.....
开机关机多重引导 终于安装好 Linux 了，也可以开机了，但是要怎样进入你的 Linux 系统呢？这里提一提第一次开机、关机如何登入你的 Linux 主机；同时并教导如何设定多重引导选单.....
档案配置与文件系统 当你第一次进入 Linux 系统中，看到的一大堆文字，在没有看过 DOS 年代的人们眼中可真是恐惧呀！这里带你了解 Linux 的路径各有何用途？ Linux 档案共有多少属性？
常用的 Linux 指令 OKOK！那接下来要怎样在 Linux 的文字接口中下达指令给 Linux 帮你执行呢？这里有最简单的察看档案、压缩解压缩、邮件传送等等指令的说明。同时包含Quota与userconf等指令的说明。
文书编辑器 vi 文书编辑软件在 Windows 下有 word ，在 Linux 下就是 vi 啦！
X-Window 的使用 如何建立 X-window 的 XFree86 server 系统？如何修改窗口呢？然后，如何让你的 NVidia 的显示适配器被 Linux 支援呢？
Linux 套件升级 如果使用了旧版的 RedHat 而要更新至新版的套件，是否需要重新安装新版的 RedHat 呢？这里告诉你如何升级你的套件！
进阶篇：编译核心 如果你是玩家的话，想对 Linux 有进一步的了解，一定要编译过核心才行！当然，如果你只是要让 Linux 当成你的 NAT 主机，那一辈子不用编译也没关系，所以这里定义为进阶篇！这里教你编辑一个适合你的 Linux 核心，并且设定新旧核心及 Windows 共存的多重引导！
例行性命令的建立 如果你每天或每个月需要在特定的某一时刻执行一些程序的话，该如何设定？
2008/08/18：将过去1999-2001年间的努力做个汇整而已！
