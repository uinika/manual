NAME         top

       slapd-ldbm - Discontinued LDBM backend to slapd

SYNOPSIS         top

       ETCDIR/slapd.conf

DESCRIPTION         top

       LDBM was the original database backend to slapd(8), and was supported
       up to OpenLDAP 2.3.  It has been superseded by the more robust BDB
       and HDB backends.

SEE ALSO         top

       slapd(8), slapd-bdb(5), slapd.backends(5).
