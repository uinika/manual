NAME         top

       libpfm_arm_ac15 - support for Arm Cortex A15 PMU

SYNOPSIS         top

       #include <perfmon/pfmlib.h>

       PMU name: arm_ac15
       PMU desc: ARM Cortex A15

DESCRIPTION         top

       The library supports the ARM Cortex A15 core PMU.

       This PMU supports 6 counters and privilege levels filtering.

MODIFIERS         top

       The following modifiers are supported on ARM Cortex A15:

       u      Measure at the user level. This corresponds to PFM_PLM3.  This
              is a boolean modifier.

       k      Measure at the kernel level. This corresponds to PFM_PLM0.
              This is a boolean modifier.

       hv     Measure at the hypervisor level. This corresponds to PFM_PLMH.
              This is a boolean modifier.
