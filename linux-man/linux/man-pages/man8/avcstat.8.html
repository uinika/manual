NAME         top

       avcstat - Display SELinux AVC statistics

SYNOPSIS         top

       avcstat [-c] [-f status_file] [interval]

DESCRIPTION         top

       Display SELinux AVC statistics.  If the interval parameter is
       specified, the program will loop, displaying updated statistics every
       interval seconds.  Relative values are displayed by default.

OPTIONS         top

       -c     Display the cumulative values.

       -f     Specifies the location of the AVC statistics file, defaulting
              to /sys/fs/selinux/avc/cache_stats.
