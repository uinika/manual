NAME         top

       lvmdbusd — LVM D-Bus daemon

SYNOPSIS         top

       lvmdbusd [--debug] [--udev]

DESCRIPTION         top

       lvmdbusd is a service which provides a D-Bus API to the logical
       volume manager (LVM).  Run lvmdbusd(8) as root.

OPTIONS         top

       --debug
              Enable debug statements

       --udev
              Use udev events to trigger updates

SEE ALSO         top

       dbus-send(1), lvm(8)
