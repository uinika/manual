NAME         top

       yum-changelog

SYNOPSIS         top

       The yum-changelog package

DESCRIPTION         top

       yum-changelog(1) is a Yum plugin for viewing package changelogs
       before/after updating.  yum will invoke yum-changelog(1) plugin if
       the --changelog option or the changelog command is used with yum.

OPTIONS         top

       --changelog
              Show changelog delta of updated packages

COMMANDS         top

       changelog
              Show changelog data of packages listed (same format as the
              list command).  The first argument is required and is either
              "all" for all the changelog entries, a date for the changelog
              entries since a specified point in time or a number for a
              given number of changelog entries.

FILES         top

       /etc/yum/pluginconf.d/changelog.conf
              The system wide configuration file. See yum-changelog.conf(5)
              for more information.

EXAMPLES         top

       yum-devel@lists.baseurl.org.  To file a bug use
       http://bugzilla.redhat.com for Fedora/RHEL/Centos related bugs and
       http://yum.baseurl.org/report for all other bugs.
