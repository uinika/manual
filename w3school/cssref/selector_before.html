CSS :before 选择器

CSS 选择器参考手册

实例

在每个 <p> 元素的内容之前插入新内容：

p:before
{ 
content:"台词：";
}

亲自试一试

浏览器支持

IE Firefox Chrome Safari Opera

所有主流浏览器都支持 :before 选择器。

注释：对于 IE8 及更早版本中的 :before，必须声明 <!DOCTYPE>。

定义和用法

:before 选择器在被选元素的内容前面插入内容。

请使用 content 属性来指定要插入的内容。

亲自试一试 - 实例

在每个 <p> 元素前面插入内容，并设置所插入内容的样式：

p:before
{ 
content:"台词：-";
background-color:yellow;
color:red;
font-weight:bold;
}
