JavaScript RegExp $ 量词

JavaScript RegExp 对象

定义和用法

n$ 量词匹配任何结尾为 n 的字符串。

语法

new RegExp("n$")

直接量语法：

/n$/

浏览器支持

所有主流浏览器都支持 $ 量词。

实例

对字符串结尾的 "is" 进行全局搜索：

var str="Is this his";
var patt1=/is$/g;

下面被标记的文本显示了表达式获得匹配的位置：

Is this his
