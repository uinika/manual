ASP.NET Helpers - 对象参考手册

Analytics 对象参考手册 (Google)

Analytics.GetGoogleHtml(webPropertyId)
呈现指定 ID 的 Google Analytics JavaScript 代码。
Analytics.GetStatCounterHtml(project, security)
呈现指定项目的 StatCounter Analytics JavaScript 代码。
Analytics.GetYahooHtml(account)
呈现指定帐户的 Yahoo Analytics JavaScript 代码。

Bing 对象参考手册

Bing.SearchBox([boxWidth])
向 Bing 传递搜索。如需规定要搜素的站点和搜索框的标题，您可以设置 Bing.SiteUrl 和 Bing.SiteTitle 属性。通常在 _AppStart 页面设置这些属性。
Bing.AdvancedSearchBox([, boxWidth] [, resultWidth] [, resultHeight] [, themeColor] [, locale])
以可选的格式在页面中显示 Bing 搜索结果。如需规定要搜素的站点和搜索框的标题，您可以设置 Bing.SiteUrl 和 Bing.SiteTitle 属性。通常在 _AppStart 页面设置这些属性。

Chart 对象参考手册

Chart(width, height [, template] [, templatePath])
对图表进行初始化。
Chart.AddLegend([title] [, name])
向图像添加图例。
Chart.AddSeries([name] [, chartType] [, chartArea] [, axisLabel] [, legend] [, markerStep] [, xValue] [, xField] [, yValues] [, yFields] [, options])
提供图表的数据点和系列特性。

Crypto 对象参考手册

Crypto.Hash(string [, algorithm]) Crypto.Hash(bytes [, algorithm])
返回指定数据的哈希值。默认算法是 sha256。

Facebook 对象参考手册

Facebook.LikeButton(href [, buttonLayout] [, showFaces] [, width] [, height] [, action] [, font] [, colorScheme] [, refLabel])
使 Facebook 用户生成到页面的连接。

FileUpload 对象参考手册

FileUpload.GetHtml([initialNumberOfFiles] [, allowMoreFilesToBeAdded] [, includeFormTag] [, addText] [, uploadText])
程序用于上传文件的 UI。

GamerCard 对象参考手册

GamerCard.GetHtml(gamerTag)
呈现指定的 Xbox 游戏者标签。

Gravatar 对象参考手册

Gravatar.GetHtml(email [, imageSize] [, defaultImage] [, rating] [, imageExtension] [, attributes])
呈现指定电邮地址的 Gravatar 图像。

Json 对象参考手册

Json.Encode(object)
将数据对象转换为 JavaScript 对象表示法 (JSON) 格式的字符串。
Json.Decode(string)
将 JavaScript 对象表示法 (JSON) 格式的数据转换为数据对象。

LinkShare 对象参考手册

LinkShare.GetHtml(pageTitle [, pageLinkBack] [, twitterUserName] [, additionalTweetText] [, linkSites])
使用指定的标题和可选 URL 来呈现社交网络连接。

ModelState 对象参考手册

ModelStateDictionary.AddError(key, errorMessage)
将错误消息添加到与指定键关联的模型状态中。
ModelStateDictionary.AddFormError(errorMessage)
将错误消息添加到与整个窗体关联的模型状态中。
ModelStateDictionary.IsValid
确定是否有任何错误消息与指定键关联。

ObjectInfo 对象参考手册

ObjectInfo.Print(value [, depth] [, enumerationLength])
呈现指定对象及任何子对象的属性名称和值。

Recaptcha 对象参考手册

Recaptcha.GetHtml([, publicKey] [, theme] [, language] [, tabIndex])
呈现 reCAPTCHA 验证测试。
ReCaptcha.PublicKey
ReCaptcha.PrivateKey
设置 reCAPTCHA 服务的公有键和私有键。通常在 _AppStart 页面设置这些属性。
ReCaptcha.Validate([, privateKey])
返回 reCAPTCHA 测试的结果。
ServerInfo.GetHtml()
呈现 ASP.NET Web Pages 的状态信息。

Twitter 对象参考手册

Twitter.Profile(twitterUserName)
呈现指定用户的 Twitter 流。
Twitter.Search(searchQuery)
呈现指定搜索文本的 Twitter 流。

Video 对象参考手册

Video.Flash(filename [, width, height])
为指定文件（可选宽度和高度）呈现 Flash 视频播放器。
Video.MediaPlayer(filename [, width, height])
为指定文件（可选宽度和高度）呈现 Windows 媒体播放器。
Video.Silverlight(filename, width, height)
为指定的 .xap 文件（必需的宽度和高度）呈现 Silverlight 播放器。

WebCache 对象参考手册

WebCache.Get(key)
从 WebCache 对象中检索指定项。
WebCache.Remove(key)
从 WebCache 对象中删除指定项。
WebCache.Set(key, value [, minutesToCache] [, slidingExpiration])
将一个项插入到 WebCache 对象。

WebGrid 对象参考手册

WebGrid(data)
初始化 WebGrid 类的新实例。
WebGrid.GetHtml()
返回用于呈现 WebGrid 实例并使用指定分页选项的 HTML 标记。
WebGrid.Pager()
返回用于为 WebGrid 实例提供指定分页支持的 HTML 标记。

WebImage 对象参考手册

WebImage(path)
加载指定路径的图像。
WebImage.AddImagesWatermark(image)
将水印图像添加到 WebImage 对象。
WebImage.AddTextWatermark(text)
在图像中添加水印文本。
WebImage.FlipHorizontal()
WebImage.FlipVertical()
水平或垂直翻转图片。
WebImage.GetImageFromRequest()
返回已使用浏览器上载的图像。
WebImage.Resize(width, height)
调整图像大小。
WebImage.RotateLeft()
WebImage.RotateRight()
将图像旋转到左侧或右侧。
WebImage.Save(path [, imageFormat])
把图像保存到指定路径。
