JavaScript

JavaScript 是因特网的标准脚本语言。

每一位开发者都应当学习 JavaScript。

请访问我们的 JavaScript 教程，以及完整的 JavaScript 参考手册。

HTML DOM

HTML 4 支持 HTML 文档对象模型(DOM)。

HTML DOM 是访问 HTML 元素的标准方法。它适用于所有浏览器。

通过 HTML DOM，您能够开发可运行于所有现代浏览器的交互式网页。

如果您希望成为专业的 web 开发者，请学习我们的 HTML DOM 教程，以及完整的 HTML DOM 参考手册。

动态 CSS

实际上并不存在名为动态 CSS 的标准。

不过，通过 JavaScript 和 HTML DOM，您能够动态地改变 HTML 元素的样式。
