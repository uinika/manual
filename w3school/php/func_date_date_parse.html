PHP date_parse() 函数

PHP Date / Time 函数

实例

返回一个包含指定日期的详细信息的关联数组：

<?php
print_r(date_parse("2016-09-25 10:45:30.5"));
?>

运行实例

定义和用法

date_parse() 函数返回包含指定日期的详细信息的关联数组。

语法

date_parse(date);
参数 描述
date 必需。规定日期（strtotime() 接受的格式）。

技术细节

返回值： 如果成功则返回包含被解析日期信息的关联数组，如果失败则返回 FALSE。
PHP 版本： 5.2+
